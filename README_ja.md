# datapack template
for ver 1.16.5 (Java)  
製作：大鶏帝国研究所  
pack_formatをいじれば他バージョンでも使えます  

# 何のデータパック？
これはデータパックではありません。  
データパックの投稿・製作テンプレートです。

# 使い方
1. ダウンロードする
2. つくる
3. アップロードする
4. わーい
